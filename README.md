🛡️ Service Guardian AI
Un Monitor de Microservicios con Inteligencia Operacional.

Este proyecto no es solo un monitor de "pings". Es un asistente diseñado para detectar fallos en servicios críticos, registrar su comportamiento y utilizar IA (LLMs) para analizar las causas raíz de los errores, sugiriendo pasos de acción inmediata para los desarrolladores.

Tech Stack: Node.js (Core), SQLite (Data), OpenAI/Claude API (Intelligence).
